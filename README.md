# LiteEdit C++

A minimalistic console text editor written in C++. This project demonstrates low-level keyboard event handling and efficient file stream management.

### Features
- **Raw Mode Input:** Instant response to key presses using `_conio.h`.
- **Navigation:** Arrow keys support to move through the text buffer.
- **File I/O:** Open existing `.txt` files or create new ones from scratch.
- **Safety:** Confirmation prompts before exiting without saving.
